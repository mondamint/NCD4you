# คู่มือการติดตั้งระบบ NCDs 4YOU บน Visual Server

แพ็คเกจนี้ประกอบด้วยโปรแกรมที่ถูกรวมรวม (Compiled) มาแล้ว พร้อมสำหรับการใช้งานบนเครื่อง Server

## สิ่งที่ต้องเตรียม
1. เครื่อง Server ที่เป็น **Windows** (แนะนำ Windows Server 2019 ขึ้นไป หรือ Windows 10/11)
2. ควรติดตั้ง **Google Chrome** หรือ Browser ที่ทันสมัย

## ขั้นตอนการติดตั้ง

1. **คัดลอกไฟล์:**
   ให้นำโฟลเดอร์ `NCDs_4YOU_Deploy` ทั้งหมด ไปวางไว้ที่ไดรฟ์ C: หรือ Desktop ของเครื่อง Server

2. **การตั้งค่า Database (Optional):**
   - โปรแกรมจะใช้ฐานข้อมูล `ncd_app.db` ที่อยู่ในโฟลเดอร์เดียวกัน
   - หากต้องการสำรองข้อมูล เพียงแค่ Copy ไฟล์ `ncd_app.db` เก็บไว้

3. **การเริ่มใช้งาน:**
   - ดับเบิ้ลคลิกไฟล์ `NCDs_4YOU.exe` (หรือชื่อไฟล์ที่คอมไพล์แล้ว)
   - หน้าต่าง Console สีดำจะปรากฏขึ้น **ห้ามปิดหน้าต่างนี้** (นี่คือ Server)
   - โปรแกรมจะเปิดหน้าเว็บขึ้นมาอัตโนมัติที่ `http://localhost:8000` (หรือ Port ที่กำหนด)

4. **การใช้งานผ่านเครื่องอื่น (Client):**
   - หากต้องการให้เครื่องอื่นในวง Lab หรือ WiFi เดียวกันเข้าใช้ง่านได้
   - ต้องทราบ **IP Address** ของเครื่อง Server (เช่น `192.168.1.100`)
   - ที่เครื่องอื่น ให้เปิด Chrome แล้วพิมพ์ `http://192.168.1.100:8000` (เปลี่ยน IP ให้ตรงกับ Server)
   - **ข้อควรระวัง:** ต้องตั้งค่า Firewall ของ Windows Server ให้เปิด Port 8000 (Inbound Rule) ด้วย

## การปิดโปรแกรม
- เพียงแค่ปิดหน้าต่าง Console สีดำ โปรแกรมจะหยุดทำงาน

## การแก้ไขข้อมูล
- ข้อมูลทั้งหมดจะถูกบันทึกใน `ncd_app.db`
- **ไม่สามารถ** แก้ไข Source Code หรือ Logic การทำงานของโปรแกรมได้ เนื่องจากถูกล็อคไว้ในไฟล์ .exe แล้ว
